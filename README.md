# 🪐 Kepler Exoplanet Archive Mission

**Authors:** Polina & Olya 

## 📝 About the Project

A full-stack Data Science project analyzing the NASA Kepler Exoplanet Search Results dataset. The mission focuses on filtering, analyzing, and visualizing confirmed exoplanets, potential candidates, and false positives.

## 🚀 Key Features

* 
**Advanced Data Analysis:** Complete Exploratory Data Analysis (EDA), data cleaning, feature engineering (e.g., Habitable Zone calculations), and rigorous hypothesis testing using Pandas, SciPy, and Plotly.


* 
**Retro NASA Dashboard:** A Streamlit web interface styled as an authentic Web 1.0 / Windows 95 mainframe, providing full access to the project's data dictionary and analytical reports.


* 
**3D Comparative Orrery:** An interactive 3D space radar that simulates planetary orbits based on Kepler's Third Law, allowing users to visually compare Hot Jupiters and Earth-like planets across different star systems.


* 
**REST API:** A FastAPI backend handling live data filtering (GET) and new planet registration (POST).
